# Week 2 Reflection

**What I learned:** I mastered asynchronous JavaScript workflows by shifting from callback chains and Promises to clean `async/await` syntax with robust `try/catch` error management. I also learned to dynamically manipulate the DOM by querying elements, rendering content from JS arrays via custom component factories, and binding custom event listeners without relying on inline attributes. Finally, I learned to leverage the browser's `localStorage` to cache API responses with TTL (Time To Live) limits and persist user states (like a dynamic favorites feature) across reloads.

**What was hard:** Implementing a robust data caching layer using `localStorage` that handles cache invalidation after 5 minutes, while synchronizing favorites toggles and real-time query filtering without causing layout reflows or duplication of event listeners, took careful design and testing.

**What I want to learn next:** I am excited to transition into Week 3 to explore professional styling systems, including advanced CSS layouts (Flexbox and Grid), responsive mobile-first structures, and the utility-first classes of TailwindCSS to bring my dynamic UIs to life with production-grade styling.
