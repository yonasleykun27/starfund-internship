# Week 2: JavaScript Deep Dive & DOM

This folder contains the daily assignments, weekly deliverables, and practical exercises for Week 2 of the StarFund Internship.

## Files Included:

* **day6.js**: Callbacks & Promises simulation modeling network latencies and mock fetch errors.
* **day7.js**: Refactored async/await functions demonstrating error catching and sequential load orchestration.
* **campaign-card.html**: Dynamic campaign card built using pure HTML + JS, rendering startup stats from a JavaScript object.
* **filter-demo.html**: Events and forms simulation implementing real-time text searches and category dropdown filters.
* **day10-landing.html**: The Week 2 Landing Page (Mini-Project). Integrates Fetch API with custom caching (5-minute TTL) and a localStorage-persisted favorites feature.
* **exercise6.html**: Practical Exercise 6 - User data cache system with a real-time countdown timer and localStorage inspector.
* **exercise7.html**: Practical Exercise 7 - Campaign countdown timer ticking down to seconds with custom date extensions.
* **exercise8.html**: Practical Exercise 8 - Dynamic pagination loading cards with skeleton layout loaders.
* **exercise9.html**: Practical Exercise 9 - Registration form checking 5 inputs in real-time with custom inline errors.
* **exercise10-broken.js**: Practical Exercise 10 - Template JS file containing 5 intentional bugs (scopes, const, unhandled promises, reference checks, await).
* **exercise10.html**: Practical Exercise 10 - Visual explanation dashboard and interactive console verifying the buggy vs. fixed code.
* **reflection.md**: Week 2 Weekly Reflection summarizing key learnings and next-step goals.
