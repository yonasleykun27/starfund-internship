Semantic Elements Used in These Cards

HTML5 semantic elements and their purpose in the campaign card
Element Role in the Card Replaces
<article> Self-contained campaign card (shareable, independent) <div class="card">
<header> Card top section — cover image + status badge <div class="card-header">
<section> Main body content group + stats block <div class="card-body">
<footer> Card bottom — progress meter, deadline, CTA <div class="card-footer">
<address> Founder attribution (contact/author info) <div class="founder">
<figure> / <figcaption> Founder avatar + name/title caption <div class="avatar-wrap">
<mark> Status badge (Active / Funded / Pending) <span class="badge">
<meter> Native funding progress indicator <div class="progress-bar">
<time> Campaign deadline (machine-readable) <span class="deadline">
<dl> / <dt> / <dd> Stats key-value pairs (Raised, Goal, Investors) <div class="stat">
<small> Category pill annotation <span class="category">
<button> Invest CTA (inherently semantic) <div class="btn"> ❌
