    Day 11 Daily Assignment Summary

    Goal: Build the StarFund page header: logo left, navigation links (Browse, How It Works, Login, Register) right — using only HTML + CSS.

    Constraints Met:

    1. No layout <div> elements used. Layout is handled entirely using <header>, <nav>, <ul>, <menu>, and <a>.
    2. CSS Flexbox is used to align logo to the left and navigation/buttons to the right.
    3. Sticky alignment using position: fixed at the top of the viewport.

    Semantic HTML Breakdown

    By using tags like <header>, <nav>, and <menu>, we describe the exact functionality of each section to the browser and assistive engines. This avoids tag pollution and improves overall code readability and SEO score.

    Sticky Positioning Demonstration

    Scroll down this page to see the header remain stuck at the top with a beautiful blur effect overlaying the content behind it.

    This is achieved using position: fixed; top: 0; left: 0; width: 100%, combined with a modern backdrop-filter: blur(12px) styling to produce a premium frosted-glass design aesthetic.
