Styled & Branded Tooltips

Using CSS gradients and brand color schemes to create premium card tooltips matching StarFund's design system.

How it Works:

  • HTML Attribute: Uses data-tooltip="label content" to hold the text content.
  • CSS Content Retrieval: The pseudo-element uses content: attr(data-tooltip) to fetch the text value dynamically.
  • Zero JavaScript: Toggle states are handled completely in the CSS compiler using the :hover and :focus-within selectors.
  • Animations: Smooth cubic-bezier transitions animate the `opacity` and `transform` properties for a premium feel.
