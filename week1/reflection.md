# Week 1 Reflection

**What I learned:** I solidified my understanding of core JavaScript fundamentals, moving from basic control flow and strict equality checks to building reusable functions. I also learned how to manipulate complex data structures using array methods like `map`, `filter`, and `reduce`, and how to model real-world entities using JavaScript objects and destructuring. 
**What was hard:** Shifting my mindset to use higher-order array methods instead of traditional `for` loops took some practice, especially understanding how `reduce` passes the accumulator value through each iteration.
**What I want to learn next:** I am excited to learn how to connect this backend logic to an actual visual interface using the DOM and asynchronous JavaScript in Week 2.
