# Week 1: Programming Fundamentals

This folder contains the daily assignments and practical tasks for Week 1 of the StarFund Internship. 

## Files Included:
* **day1.js**: Basic `for` loops and an `if/else` voting eligibility checker.
* **day2.js**: Variable data types, a grade calculator, and a login checker using logical `&&` operators.
* **day3.js**: Functions, including a prime number checker, multiplication tables, and a campaign funding calculator.
* **day4.js**: Higher-order array methods (`map`, `filter`, `reduce`) used to process a pipeline of campaign data.
* **day5.js**: Object creation and destructuring for a StarFund startup model.
* **starfund-data.js**: The Week 1 Mini-Project. Contains an array of startup objects and functions to filter by status, calculate total funding, and find the most funded startup.
